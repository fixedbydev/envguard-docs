# Concepts Overview

Learn about the core concepts behind envguard and how it validates your environment variables at runtime using Zod schemas.

## Key Concepts

- **Schema-first validation** -- Define your environment schema using Zod, and envguard validates at startup.
- **Type safety** -- Get full TypeScript inference for all your environment variables.
- **Fail-fast** -- Invalid or missing variables cause clear errors before your app starts.
- **Composable schemas** -- Combine and extend schemas across packages and environments.
