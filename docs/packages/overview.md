# Packages

The envguard ecosystem is composed of several packages.

## Core Packages

| Package | Description |
|---------|-------------|
| `@stacklance/envguard-core` | Core validation library with Zod schemas |
| `@stacklance/envguard-cli` | Command-line interface for envguard |
| `create-envguard` | Project scaffolding tool |

## Integration Packages

| Package | Description |
|---------|-------------|
| `envguard-vscode` | VS Code extension |
| `envguard-action` | GitHub Action |
| `envguard-badge` | Badge service |
